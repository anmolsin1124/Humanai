## 🌌 Overview

Presence AI is a next-generation conversational system built to create meaningful, human-like interactions.

Unlike traditional chatbots that only respond to inputs, Presence AI understands **context, tone, and emotional signals**. It adapts its behavior dynamically and builds continuity using a memory-driven approach.

This creates an experience where conversations feel natural, evolving, and personal.

## ✨ Features

* 🧠 **Emotion-Aware Responses**
  Detects hidden feelings and intent behind messages

* 🔁 **Memory-Based Interaction**
  Remembers past conversations and recurring patterns

* 🎭 **Adaptive Personality Modes**
  Switches tone (deep, soft, chill) based on user

* 💬 **Human-Like Conversation Flow**
  Uses pauses, reflections, and layered questions

* 🎨 **Minimal & Modern UI**
  Clean design focused on readability and experience

## 🚀 Getting Started

```bash

npm install
npm run dev
```

## 🔐 Environment Variables

Create a `.env` file:

```env
API_KEY=your_api_key_here
```

## 🌍 Deployment (Vercel)

1. Push project to GitHub
2. Import repo in Vercel
3. Add environment variables
4. Click deploy
